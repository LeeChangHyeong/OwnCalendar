# 💻 수강생 관리 프로그램
내일배움캠프의 첫 번째 개인 Spring 프로젝트 

## 🔨 개발환경
- **Java 21**
- **SDK : openjdk-21**
- **IDE : Intellij IDEA 2023.3.4**
- **Tools : Github, Git**
<br>

## 🔔 주요기능
### 스케쥴 관리
**💡 회원가입** <br>
**💡 로그인** <br>
**💡 스케쥴 등록** <br>
**💡 스케쥴 목록 조회 (전체 조회, 특정 스케쥴 조회)** <br>
**💡 스케쥴 삭제** <br>
**💡 댓글 등록** <br>
**💡 댓글 삭제** <br>
