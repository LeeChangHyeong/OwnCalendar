# 💻 수강생 관리 프로그램
내일배움캠프의 첫 번째 개인 Spring 프로젝트 

## 🔨 개발환경
- **Java 21**
- **SDK : openjdk-21**
- **IDE : Intellij IDEA 2023.3.4**
- **Tools : Github, Git**
<br>
